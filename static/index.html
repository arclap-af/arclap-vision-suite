Train a custom YOLO model

      
Upload a CVAT export (zip) — the kind that comes out of CVAT's "Ultralytics YOLO" export option. The Suite extracts the dataset, fine-tunes a YOLO model on it, and registers the resulting .pt in the Models tab automatically.

    

    

      
      

        
        
Drop a CVAT export zip or browse

        
Must contain a data.yaml with names: and train: / val: paths · max 5 GB

      

    

    

  


  

    

      
Training configuration

      
Sensible defaults for ~500 labelled frames. Bigger datasets benefit from more epochs.

    

    

      
      
      
      
      
      
    

    

      
      Training time scales with dataset size × epochs. Plan for minutes (small dataset, n model) to hours (large dataset, x model).

About Arclap Vision Suite

      
    

    

      
Tools

      
    
        
  • Wizard — drag a video / pick a folder / paste a URL, choose a cleanup goal, preview, run.
    • 
        
  • Models — upload any YOLOv8 / YOLOv11 .pt file or one-click install a suggested model.
    • 
        
  • Train — fine-tune a YOLO model on a CVAT export.
    • 
        
  • Live RTSP — connect to any IP camera and run YOLO live.
    • 
        
  • History — every past job with re-run / verify / audit export.
    • 
        
  • Projects — group jobs, save recipes, retention cleanup.
    • 
      

      
Keyboard shortcuts

      
    
        
  • Cmd/Ctrl + Enter — run the wizard's full job
    • 
        
  • ? — open this help
    • 
        
  • Esc — close this dialog
    • 
      

      
Environment variables

      
    
        
  • ARCLAP_SMTP_HOST / _PORT / _USER / _PASS / _FROM — enable email notifications on job completion.
    • 
      

      
Privacy

      
Everything runs locally. Videos, frames, models, audit reports never leave the box. The web UI binds to 127.0.0.1 only.

Everything runs on your machine — no cloud, no telemetry. The bell icon top-right will ping you when a job finishes.
